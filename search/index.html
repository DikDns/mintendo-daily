Pembelajaran Berbasis Proyek

Pembelajaran berbasis proyek adalah pembelajaran yang dilakukan dengan cara membuat suatu proyek.

JavaScript Dasar

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat website interaktif.

HTML Dasar

HTML adalah bahasa markup yang digunakan untuk membuat struktur website.

CSS Dasar

CSS adalah bahasa style sheet yang digunakan untuk mengatur tampilan website.

React Dasar

React adalah library JavaScript yang digunakan untuk membuat website interaktif.
